# HR-Analytics-dashboard
# Power BI HR Analytics Dashboard Project
Project Overview

This project focuses on analyzing employee data to identify workforce trends, employee attrition patterns, job satisfaction levels, departmental performance, and demographic insights using an interactive Power BI dashboard.

The dashboard helps HR teams and business managers make data-driven decisions related to employee retention, workforce planning, performance monitoring, and organizational improvement.

# Objectives
Analyze employee attrition and retention trends
Monitor workforce demographics and education background
Evaluate employee satisfaction across job roles
Identify departments with high attrition rates
Provide interactive HR insights using Power BI visualizations
